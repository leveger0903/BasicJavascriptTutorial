數學方法

ES5 數學方法

參考網址: https://www.w3schools.com/jsref/jsref_obj_math.asp

abs

返回絕對值

var num = -100;
num = Math.abs(num); // 100

ceil

無條件進位

var num = 99.3;
num = Math.ceil(num); // 100

floor

無條件捨去

var num = 100.7;
num = Math.floor(num); // 100

round

數字四捨五入

var num = 100.3;
num = Math.round(num); // 100

max

取得最大值

var array = [1, 5, 9];
var num = Math.max(...array); // 9

min

取得最小值

var array = [1, 5, 9];
var num = Math.min(...array); // 1

random

亂數取得 0 ~ 1 內的數字

var num = Math.random();
